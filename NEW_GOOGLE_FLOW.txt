════════════════════════════════════════════════════════
         📋 HOW TO CONNECT (3 SIMPLE STEPS)
════════════════════════════════════════════════════════

STEP 1: Get Google Credentials
────────────────────────────────
1. Go to: https://console.cloud.google.com/
2. Create project (or select existing)
3. Enable: Google Drive API & Google Docs API
4. Create OAuth Client ID (Web application)
5. Redirect URI: http://localhost:8000/api/oauth/google/callback
6. Copy: Client ID & Client Secret

STEP 2: Configure in Settings
────────────────────────────────
1. Go to: http://localhost:3000/settings
2. Find: "Google OAuth Configuration"
3. Enter:
   - Client ID (ends with .apps.googleusercontent.com)
   - Client Secret (starts with GOCSPX-)
   - Redirect URI (pre-filled)
4. Click: "Save OAuth Credentials"

STEP 3: Connect Google Account
────────────────────────────────
1. In Settings, find: "Google Workspace Connection"
2. Click: "Connect Google Workspace"
3. Authorize in Google popup
4. Done! ✅
