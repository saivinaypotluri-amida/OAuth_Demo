════════════════════════════════════════════════════════
         STEP 1: RESTART BACKEND
════════════════════════════════════════════════════════

In backend terminal:
  Press CTRL+C
  Run: python main.py

════════════════════════════════════════════════════════
         STEP 2: INSTALL & RUN NGROK
════════════════════════════════════════════════════════

Download: https://ngrok.com/download

Or install:
  npm install -g ngrok

Run in NEW terminal:
  ngrok http 8000

Copy the HTTPS URL:
  https://abc123.ngrok.io

════════════════════════════════════════════════════════
         STEP 3: CONFIGURE SLACK APP
════════════════════════════════════════════════════════

1. Go to: https://api.slack.com/apps

2. Select your app (or create one)

3. Click "Event Subscriptions"

4. Toggle ON

5. Request URL:
   https://YOUR_NGROK_URL.ngrok.io/api/slack/events

6. Wait for ✅ "Verified"

7. Add Bot Events:
   - message.channels
   - message.im
   - app_mention

8. Save Changes → Reinstall app

════════════════════════════════════════════════════════
         STEP 4: TEST IN SLACK
════════════════════════════════════════════════════════

1. In Slack, add bot to channel:
   /invite @YourBot

2. Send message:
   "Hello bot!"

3. Bot responds automatically! ✅
