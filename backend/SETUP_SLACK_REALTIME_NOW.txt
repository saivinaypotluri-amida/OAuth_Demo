════════════════════════════════════════════════════════
         STEP 1: RESTART BACKEND (30 sec)
════════════════════════════════════════════════════════

In backend terminal:
  CTRL+C to stop
  python main.py

Verify: "Application startup complete." ✅

════════════════════════════════════════════════════════
         STEP 2: INSTALL & RUN NGROK (2 min)
════════════════════════════════════════════════════════

Download from: https://ngrok.com/download

Or install:
  npm install -g ngrok

Run in NEW terminal:
  ngrok http 8000

Copy the HTTPS URL shown:
  https://abc123xyz.ngrok.io
  
Keep this terminal open!

════════════════════════════════════════════════════════
    STEP 3: CONFIGURE SLACK EVENTS (5 min)
════════════════════════════════════════════════════════

1. Go to: https://api.slack.com/apps

2. Select your Slack app

3. Click "Event Subscriptions" (sidebar)

4. Toggle "Enable Events" ON

5. Request URL: 
   https://YOUR_NGROK_URL.ngrok.io/api/slack/events
   
   (Replace YOUR_NGROK_URL with your actual ngrok URL)

6. Wait for ✅ "Verified"

7. Under "Subscribe to bot events", add:
   → message.channels
   → message.im
   → app_mention

8. Click "Save Changes"

9. Click "reinstall your app" → Allow

════════════════════════════════════════════════════════
         STEP 4: TEST IN SLACK (1 min)
════════════════════════════════════════════════════════

1. Open Slack

2. Go to a channel

3. Add bot: /invite @YourBot

4. Send message:
   "Hello bot! What can you do?"

5. Bot responds automatically! 🎉

════════════════════════════════════════════════════════
            🐛 TROUBLESHOOTING
════════════════════════════════════════════════════════

Bot doesn't respond?
  → Check Azure OpenAI is configured
  → Check bot is added to channel
  → Check backend logs for errors

Verification fails?
  → Ensure backend is running
  → Ensure ngrok is running
  → Check URL is correct
  → Try: curl https://your-url.ngrok.io/health

Bot responds twice?
  → Only run one ngrok instance
  → Bot ignores its own messages (already coded)
